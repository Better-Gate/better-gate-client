# Better Gate Client Releases

This repository is the public release channel for Better Gate Client.

Download the latest signed desktop build from the Releases page:

https://github.com/Better-Gate/better-gate-client/releases

Existing Better Gate Client installations use this repository for automatic updates. The application source code is maintained privately by Better Gate.

## Security

Better Gate Client release artifacts are produced by the Better Gate team and signed for Tauri updater verification. Do not install packages from unofficial mirrors.

## Acknowledgements

Better Gate Client was originally based on the open-source cc-switch project. We are grateful to the cc-switch maintainers and contributors for their work.
